# 阶段 2 验收清单（Agent Skills）

## A. 能力可用性

1. `monitor_keywords`
- 能创建关键词规则
- 能触发一次扫描
- 能返回该规则命中的热点列表

2. `discover_trends`
- 能触发扫描
- 能返回趋势聚合结果

3. `validate_event`
- 能调用 `/api/skills/validate`
- 能返回标准评分字段与理由

## B. 结构化输出

- 输出为 JSON
- 字段包含 `relevance_score/authenticity_score/novelty_score/importance_score/noise_score`

## C. 运行验收

- `node agent-skills/skills-cli.mjs discover_trends "AI 编程"`
- `node agent-skills/skills-cli.mjs monitor_keywords "openai model release" high`
- `node agent-skills/skills-cli.mjs validate_event "openai model release" "OpenAI发布新模型" "OpenAI announced a new model update" "https://openai.com/blog"`

以上命令均应成功执行并返回 JSON。
